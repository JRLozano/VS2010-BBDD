Archivo

Aquí se incluirán todas las diferentes categorías que hay dentro del menú Archivo.

Abrir

Desde el menú Archivo/Abrir, se puede abrir una nueva imagen para poder manipularse.

Guardar

Desde el menú Archivo/Guardar, se puede guardar una nueva imagen en diferentes formatos (PNG, TIFF, JPG, etc.).

Salir

Desde el menú Archivo/Salir, se cierra el programa, aunque no directamente, se mostrará un aviso de si realmente se quiere cerrar la aplicación
